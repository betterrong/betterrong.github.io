# Linux是什么与如何学习

作者：溶 时间：2020年07月15日

# 0x00 Linux是什么

# 1 Linux是什么？操作系统、应用程序？

  • 每种操作系统都是在它专门的硬件架构上面运行。

# 2 Linux的历史

  • Linux是由Linus Toevalds先生于1991年编写。
  • MIT发展了兼容分时系统，它可以让多个用户在某一时间段内分别使用CPU的资源。
  • 1969年Ken Thompson 费时四个星期，用汇编语言写出了Unics操作系统 ，两给重要的概念：
    • 所有的系统设备或程序都是文件
    • 不管程序本身还是附属文件，所写的程序只有一个目的，有效地完成目标。
  • 1973年：UNIX正式诞生，Ritchie等人用C写出来的第一个UNIX内核。
  • 1977年：UNIX分支：BSD的诞生
  • 1979年：System V架构与版权声明
  • 1979年：AT&T收回UNIX版权，严禁向学生提供UNIX源代码。
  • 1984年：Tanenbaum教授开始编写MInix这个Unix-like内核。
  • 1984年：GNU计划和FSF基金会成立，Richard Mathew Stalllman 发起GNU计划
    • 目的：建立一个自由的、开放的UNIX操作系统。
    • 编写GNU C Complier
    • 编写Emacs编辑器
    • 编写BashShell
  • 1985年：草拟GPL通用公共许可证，并将之称为copyleft
  • 1991年：Linus Torvalds发布了自己使用bash、gcc等GNU工具便写的内核程序。
  • 开放源代码
  • 闭源软件

# 0x01 LInux的发展

  • POSIX 可移植操作系统接口
  • 1994年LInux和虚拟团队完成了Linux的内核正式版，还加入了x-window system的支持。

# 1 Linux的内核版本

  • Linux内核版本编号类似

    3.10.0-123.e17.x86-64
主版本.次版本.发布版本-修改版本
    1
    2

  • 奇数、偶数版本分类（2.6以前，内核发展方向分两类）

    • 主、次版本为奇数：开发中版本
    • 主、次版本为偶数：稳定版本

  • 主线版本、长期维护版本

# 2 Linux发行版

  • Linux发行版主要分为两大家族：
    • 使用RPM方式安装软件的系统
    • 使用dpkg方式安装软件的系统
