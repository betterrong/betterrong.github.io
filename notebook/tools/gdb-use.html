# GDB

GDB 功能：

  • 运行程序，设置影响程序运行的参数。
  • 让程序在指定的条件下停止运行。
  • 在程序停止时检查所有参数的情况。
  • 根据指定条件改变程序的运行。

# 0x01 快速入门

使用 gcc 编译程序时，加上 '-g' 选项。

# 1.1 进入gdb

gdb a.out
1

# 1.2 查看文件

(gdb) l
1

# 1.3 设置断点

（gdb)b 23# 在第23行设置断点
1

# 1.4 查看断点处情况看

(dgb)info b
1

# 1.5 运行代码

(gdb) r
1

程序运行到断点处就会停止。

# 1.6 查看变量值

(gdb) p x#查看x变量的值
1

GDB 在显示变量值时都会在对应值之前加上 $N 标记，它是当前变量值的引用标记，所以以后再次引用直接写作 $N 即可。

# 1.7 观察变量

观察一个变量的变化情况。

(gdb) watch n
1

# 1.8 单步运行

(gdb) n
1

# 1.9 程序继续运行

(gdb) c
1

# 1.10 退出GDB

(gdb) q
1
